Condicionales en Python

En Python, podemos usar la estructura `if...elif...else` para ejecutar diferentes bloques de código según las condiciones.

x = 20
if x > 30:
    print("x es mayor que 30")
elif x > 10:
    print("x está entre 10 y 30")
else:
    print("x es menor o igual a 10")
